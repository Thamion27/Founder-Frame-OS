# Example Product Forge Output

## Product Name

Local AI Growth Readiness Diagnostic

## User

Local service-business owners with outdated websites and weak AI discoverability.

## Promise

Show the business owner where AI can create practical modernization value without overwhelming them.

## MVP Scope

- Website URL intake
- Manual/AI-assisted diagnostic
- Readiness score
- Opportunity map
- 3-step implementation plan
- Consultation CTA

## Not in MVP

- Full SaaS dashboard
- Automated billing
- Multi-user accounts
- Complex analytics
- Custom LLM training

## Proof Test

Sell or deliver 3 diagnostics manually before automating.
