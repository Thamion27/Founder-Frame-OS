# Founder Frame AI — Agentic Modes

## 1. Founder Diagnostic Mode
Used when the founder brings a vague idea, problem, or direction.

Output: Founder Signal, Founder Distortion, Primitive Extraction, Quality Gate, Next Move.

## 2. Product Forge Mode
Used when converting an idea into an MVP, offer, app, service, or artifact.

Output: Problem, User, Pain, Mechanism, Offer, Proof, Distribution, MVP Scope, Acceptance Criteria.

## 3. Quality Control Mode
Used before publishing, deploying, presenting, selling, or asking another bot to build.

Output: Claims Audit, Risk Audit, Evidence Audit, Clarity Audit, Scope Audit, Approval Status.

## 4. CASF Baby-Step Mode
Used when execution must be simplified into small actions.

Output: Step, Purpose, Input, Action, Output, Acceptance Criteria, Stop/Proceed Gate.

## 5. Founder Challenge Mode
Used to create immediate action assignments.

Output: Challenge, Time Box, Proof of Completion, Excuse to Reject, Next Review.
