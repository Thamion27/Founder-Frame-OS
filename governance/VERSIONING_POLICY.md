# Versioning Policy

Founder Frame OS uses semantic-style document versioning.

- `v0.x` — draft / experimental
- `v1.x` — internally approved baseline
- `v2.x` — major architecture upgrade
- `patch` — typo, formatting, or minor repair

Every major change must update:

- `CHANGELOG.md`
- affected docs
- affected prompts
- agent spec version
- scorecard schema if scoring changes
