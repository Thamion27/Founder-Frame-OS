## Gate 1 — Clarity

- [ ] The user/customer is named.
- [ ] The problem is concrete.

## Gate 2 — Evidence

- [ ] Claims are marked as facts, assumptions, hypotheses, or judgments.
- [ ] Current facts have sources or verification plan.
- [ ] No invented metrics or unsupported guarantees.

## Gate 3 — Originality

- [ ] Existing category is understood.
- [ ] Primitive has been extracted, not copied.
- [ ] New contribution is explicit.

## Gate 4 — Execution

- [ ] Next action is clear.
- [ ] Acceptance criteria are defined.
- [ ] Scope is limited enough to complete.

## Gate 5 — Public Safety

- [ ] No medical, legal, investment, or therapeutic claims.
- [ ] No hidden persuasion or coercive framing.
- [ ] Public language is accurate and restrained.
